taxonomy. Return one to four taxonomy cell allocations. Use one allocation for
a single-scope item and multiple allocations only when the item clearly bundles
separate scopes. Allocation fractions should sum to 1.0. Use only active cell
codes supplied in the input. Do not invent taxonomy cells and do not do
arithmetic beyond allocation fractions.
