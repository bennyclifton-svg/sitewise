## Conventions (loader contract)

- taxonomy.yaml `meta.defaults` apply to any omitted cell field.
- `grp` is derived from the code prefix via the `groups` table.
- Conditional applicability lives in expectations.yaml ONLY (single source
  of truth); `taxonomy.applicability` is reserved for future use.
- Loaders are idempotent upserts keyed on stable codes (`cells.code`,
  `rules.rule`, `benchmark_key×state×region×build_type×spec_level×metric`).
- Load order: taxonomy → synonyms → expectations → benchmarks → report_language.

## Regeneration

```
python tools/expand_synonyms.py    # synonyms.base.csv -> synonyms.seed.csv
python tools/build_benchmarks.py   # -> benchmarks.seed.csv
python tools/validate.py           # cross-reference integrity; run in CI
```
