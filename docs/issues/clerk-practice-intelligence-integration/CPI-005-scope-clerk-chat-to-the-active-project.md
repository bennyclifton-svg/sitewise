# Scope Clerk Chat To The Active Project

## Parent

Clerk Practice Intelligence Integration PRD.

## What to Build

Make Clerk chat project-scoped when launched from a SiteWise project cockpit. The completed slice should attach project context to a thread or chat turn, filter retrieval to the active project evidence plus shared SiteWise platform knowledge, and return cited answers that make the project/source boundary visible.

## Acceptance Criteria

- [x] A chat started from a project cockpit carries active project scope to the backend.
- [x] Retrieval includes active project evidence.
- [x] Retrieval includes shared SiteWise platform knowledge where relevant.
- [x] Retrieval excludes other project evidence by default.
- [x] Citations identify whether each source is project evidence or platform knowledge.
- [x] Existing global or legacy chat behavior remains available outside the project cockpit.
- [x] Backend tests prove project-scoped retrieval does not leak another project's evidence.
- [x] Manual verification covers asking a project question and inspecting citations.
