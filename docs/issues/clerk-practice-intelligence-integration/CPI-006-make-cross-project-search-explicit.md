# Make Cross-Project Search Explicit

## Parent

Clerk Practice Intelligence Integration PRD.

## What to Build

Add an explicit cross-project search mode so broad retrieval cannot happen silently from inside a project cockpit. The completed slice should make project-scoped retrieval the default and require a clear user-visible control or command before Clerk searches across multiple projects.

## Acceptance Criteria

- [x] Project cockpit chat defaults to active-project retrieval.
- [x] Cross-project retrieval requires an explicit user action or mode.
- [x] The UI labels cross-project mode clearly when it is active.
- [x] Backend request handling distinguishes project-scoped and cross-project retrieval.
- [x] Cross-project mode still respects organisation/user permissions.
- [x] Tests cover default scoping and explicit broad search behavior.
