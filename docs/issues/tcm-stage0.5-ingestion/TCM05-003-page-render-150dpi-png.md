## What to Build

Add `render_page_png(pdf_bytes, *, page_no, dpi) -> bytes` to `tender/services/pdf.py`.
Every page is rendered to PNG at 150 DPI at ingest so the QA console can show
source-region highlights without re-rendering (PRD §7.4). Pure function; PNG bytes out.

## Design

```python
def render_page_png(pdf_bytes: bytes, *, page_no: int, dpi: int) -> bytes:
    document = fitz.open(stream=pdf_bytes, filetype="pdf")
    try:
        page = document.load_page(page_no - 1)   # page_no is 1-based
        return page.get_pixmap(dpi=dpi).tobytes("png")
    finally:
        document.close()
```

## TDD

Add a render test that parses the PNG header to assert dimensions. A default PyMuPDF
page is US Letter (612×792 pt = 8.5×11 in), so at 150 DPI it is **1275×1650 px**.
Parse width/height from PNG bytes (`data[:8] == b"\x89PNG\r\n\x1a\n"`,
`struct.unpack(">II", data[16:24])`). See plan Task 2, Step 1 for the exact test.
