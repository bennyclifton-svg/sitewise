# TCM Stage 0.5 — Ingestion, Classification & Extraction Wiring Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make the per-quote front half of the TCM pipeline actually run — implement `ingest_document` (PDF → OCR-where-needed → per-page PNG render → `tender_pages`), implement `classify_document` (LLM doc-type adjudication), wrap the already-real `extraction.py` into an `extract_line_items` worker handler, register all three in the worker, and auto-chain them so one upload flows `ingest → classify → extract → embed` without manual button clicks.

**Architecture:** The TCM pipeline is a poll-loop worker over `tender_jobs` (see [worker.py](../../backend/tender/worker.py)). Each stage is an `async def handler(session, job, *, deps=None)` registered in `worker.HANDLERS`, idempotent and checkpointed. This plan fills the three missing/stubbed front-half handlers and wires the sequential chain from PRD §7.2. **TCM ingestion is deliberately separate from Clerk's RAG `backend/ingest/` pipeline** (PRD §7.1, line 154): Clerk chunks for chat retrieval; TCM renders page images + per-page text for schema-oriented vision extraction. They share only the upload/storage layer (`app.storage.project_files`). No reuse of `backend/ingest/` — confirmed: it has no OCR and no page rendering.

**Tech Stack:** Python 3.12, async SQLAlchemy, FastAPI, PyMuPDF (`fitz`, already a dependency) for text extraction + page render, `pytesseract` + Tesseract binary for OCR (new dep), Pillow for image handling (new dep), OpenAI Responses API via the existing `tender.llm` client abstraction, pytest (TDD, repo style with mocked session/storage/LLM).

## Key design decisions (read before starting)

1. **OCR approach — render-then-OCR-the-image (pytesseract), not `ocrmypdf`.** The M1 skeleton plan ([2026-06-12-tcm-m1-skeleton.md:212](2026-06-12-tcm-m1-skeleton.md) Task 8) suggested `ocrmypdf --skip-text`. We diverge deliberately: we already render every page to a 150-DPI PNG (required for the QA console regardless), so for OCR-candidate pages we run `pytesseract.image_to_data` on that same PNG. This yields **per-page text *and* a per-page mean word confidence** in one pass — `ocrmypdf` makes per-page confidence awkward and produces a second PDF we don't need. Both approaches require the Tesseract binary on the host/VPS. If the executor prefers `ocrmypdf` for any reason, flag it before Task 3.

2. **Auto-chaining is in scope.** Currently the only enqueue-next coupling is `run_expectations → infer_silence` ([expectations.py:254](../../backend/tender/services/expectations.py#L254)); per-quote stages are triggered manually via `POST /quotes/{id}/retry/{stage}`. That is why testing required clicking each button. PRD §7.2 specifies the per-quote stages "run in sequence". This plan makes each front-half handler enqueue the next on success and advance `quote.stage`: `ingest_document → classify_document → extract_line_items → embed_items`. (`embed_items → map_items` chaining is **out of scope** here — left as a follow-up so this plan stays a reviewable vertical slice ending at embedded line items.)

3. **PDF-only core; XLSX/DOCX deferred to the optional final task.** PRD §7.4 wants XLSX/DOCX → LibreOffice-rendered PDF. That is heavy infra (LibreOffice headless) and the pilot documents are PDFs. Non-PDF inputs keep the existing `ingest_status='unsupported_format'` behaviour (M1 Decision 5). Task 9 adds office conversion and is explicitly optional.

4. **No Alembic migration needed.** Every column this plan writes already exists: `tender_documents.content_hash / doc_type / classification_confidence / ocr_applied / page_count / ingest_status`, and the whole `tender_pages` table ([models.py:249](../../backend/tender/models.py#L249)). Task 1 verifies this rather than assuming.

5. **Reuse existing seams, do not reinvent:**
   - `app.storage.project_files.download_project_file(*, storage_key)` / `upload_project_file(*, storage_key, content, filename)` — sync, wrap in `asyncio.to_thread` (router already does, [router.py:132](../../backend/tender/router.py#L132)).
   - `tender.services.mapping._context_for_quote(session, quote_id)` → `ProjectContext` ([mapping.py:653](../../backend/tender/services/mapping.py#L653)). It is a private import; Task 5 promotes it to a shared `tender/services/context.py` (DRY — `silence.py` already needs it too).
   - `tender.services.extraction.extract_line_items(*, pages, context, llm_client, stated_total_cents, ...)` → `ExtractionResult` ([extraction.py:47](../../backend/tender/services/extraction.py#L47)). Pure, already tested; the new handler only loads inputs and persists outputs.
   - `tender.llm.openai_client.AsyncOpenAITenderClient` with `.adjudicate(question, choices, evidence, context, *, prompt_version, model_key)` for classification.
   - `tender.services.jobs.enqueue(session, *, kind, comparison_id, quote_id, payload)` for chaining.

# backend/tender/services/pdf.py
"""Pure PDF helpers for TCM ingestion: text-layer extraction and OCR detection.

No DB, storage, or network — kept side-effect-free so the ingestion handler
stays a thin orchestrator over testable units (PRD §7.4).
"""

from __future__ import annotations

from dataclasses import dataclass

import fitz  # PyMuPDF


@dataclass(frozen=True)
class PageExtract:
    page_no: int          # 1-based
    text: str             # text-layer content (may be empty for image-only pages)
    text_chars: int       # len(text.strip())
    page_area: float      # width * height in PDF points


def extract_pages(pdf_bytes: bytes) -> list[PageExtract]:
    document = fitz.open(stream=pdf_bytes, filetype="pdf")
    try:
        pages: list[PageExtract] = []
        for index, page in enumerate(document, start=1):
            text = page.get_text("text").strip()
            rect = page.rect
            pages.append(
                PageExtract(
                    page_no=index,
                    text=text,
                    text_chars=len(text),
                    page_area=float(rect.width * rect.height),
                )
            )
        return pages
    finally:
        document.close()


def is_ocr_candidate(page: PageExtract, *, threshold: float) -> bool:
    """A page is an OCR candidate when its text density (chars per unit page
    area) falls below ``threshold``. Matches ``tender_ocr_text_density_threshold``.
    """
    if page.page_area <= 0:
        return True
    density = page.text_chars / page.page_area
    return density < threshold
```

# add to backend/tender/services/pdf.py

def render_page_png(pdf_bytes: bytes, *, page_no: int, dpi: int) -> bytes:
    """Render a 1-based page to PNG bytes at the given DPI (PRD §7.4)."""
    document = fitz.open(stream=pdf_bytes, filetype="pdf")
    try:
        page = document.load_page(page_no - 1)
        pixmap = page.get_pixmap(dpi=dpi)
        return pixmap.tobytes("png")
    finally:
        document.close()
```

## Task 3: `ingest_document` handler (the core of Stage 0.5)

Replaces the stub at [ingestion.py:10](../../backend/tender/services/ingestion.py#L10). Orchestrates download → hash dedupe → format gate → per-page density/OCR → render + persist per page (checkpoint unit) → finish. OCR is injected so tests never need Tesseract.

**Files:**
- Rewrite: `backend/tender/services/ingestion.py`
- Test: `backend/tests/tender/test_ingestion.py`

**Design notes (build into the implementation):**
- Handler signature: `async def ingest_document(session, job, *, downloader=None, uploader=None, ocr=None) -> None`. Defaults wire the real storage + pytesseract OCR; tests inject fakes.
- `document_id` comes from `job.payload["document_id"]`; load the `TenderDocument`.
- **Dedupe:** `content_hash` is already set at upload ([router.py:145](../../backend/tender/router.py#L145)). If another *ingested* `tender_documents` row in the same `quote_id` shares this hash and is a different id → set this row `ingest_status='duplicate'`, advance, return.
- **Format gate:** non-`application/pdf` mime (and filename not `.pdf`) → `ingest_status='unsupported_format'`, return (no chain — nothing to extract).
- **Per page:** density via Task 1; OCR candidates get `ocr(png_bytes) -> OcrResult(text, mean_confidence)`. Non-candidates use the text layer (`ocr_confidence=None`).
- **Escape hatch:** if any page was OCR'd and the mean OCR confidence across OCR'd pages `< tender_ocr_min_confidence` → `ingest_status='manual_transcription_required'`, do **not** persist junk pages, do **not** chain, return.
- **Checkpointed persist:** for each page not already in `tender_pages` (resume via `UNIQUE (document_id, page_no)`), render PNG, upload to `tender/comparisons/{comparison_id}/quotes/{quote_id}/documents/{document_id}/pages/page-{n:04d}.png`, insert `tender_pages` row, `await session.flush()` per page so a crash resumes.
- **Finish:** set `page_count`, `ocr_applied`, `ingest_status='ingested'`, advance `quote.stage='classify_document'`, and `jobs.enqueue(kind='classify_document', comparison_id, quote_id, payload={'document_id': str(document_id)})`. (The chain enqueue lives here; wiring test is Task 6.)

# backend/tender/services/ingestion.py
"""Document ingestion stage (PRD §7.4 / §9.1).

Downloads the uploaded file, dedupes by content hash, gates non-PDF formats,
detects per-page OCR need, renders every page to a 150-DPI PNG, persists
``tender_pages`` rows page-by-page (checkpoint/resume), and chains to
``classify_document``. OCR/storage are injected so the unit tests never touch
Tesseract or Supabase.
"""

from __future__ import annotations

import asyncio
import uuid
from collections.abc import Callable
from dataclasses import dataclass

from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from app.config import settings
from app.logging import get_logger
from tender.models import TenderDocument, TenderPage, TenderQuote
from tender.services import jobs
from tender.services.pdf import extract_pages, is_ocr_candidate, render_page_png

log = get_logger(__name__)


@dataclass(frozen=True)
class OcrResult:
    text: str
    mean_confidence: float


Downloader = Callable[..., bytes]
Uploader = Callable[..., str]
Ocr = Callable[[bytes], OcrResult]


def _default_downloader(*, storage_key: str) -> bytes:
    from app.storage.project_files import download_project_file

    return download_project_file(storage_key=storage_key)


def _default_uploader(*, storage_key: str, content: bytes, filename: str) -> str:
    from app.storage.project_files import upload_project_file

    return upload_project_file(storage_key=storage_key, content=content, filename=filename)


def _default_ocr(png_bytes: bytes) -> OcrResult:
    import io

    import pytesseract
    from PIL import Image

    image = Image.open(io.BytesIO(png_bytes))
    data = pytesseract.image_to_data(image, output_type=pytesseract.Output.DICT)
    words = [w for w, c in zip(data["text"], data["conf"]) if w.strip() and int(c) >= 0]
    confs = [int(c) for c in data["conf"] if int(c) >= 0]
    mean = (sum(confs) / len(confs) / 100.0) if confs else 0.0
    return OcrResult(text=" ".join(words), mean_confidence=mean)


def _is_pdf(document: TenderDocument) -> bool:
    return document.mime_type == "application/pdf" or document.original_filename.lower().endswith(".pdf")


async def ingest_document(
    session: AsyncSession,
    job,
    *,
    downloader: Downloader | None = None,
    uploader: Uploader | None = None,
    ocr: Ocr | None = None,
) -> None:
    downloader = downloader or _default_downloader
    uploader = uploader or _default_uploader
    ocr = ocr or _default_ocr

    document_id = uuid.UUID(job.payload["document_id"])
    document = await session.get(TenderDocument, document_id)
    if document is None:
        raise ValueError(f"ingest_document: document {document_id} not found")

    # Dedupe within the quote.
    if document.content_hash is not None:
        dupe = await session.execute(
            select(TenderDocument.id).where(
                TenderDocument.quote_id == document.quote_id,
                TenderDocument.content_hash == document.content_hash,
                TenderDocument.id != document.id,
                TenderDocument.ingest_status == "ingested",
            )
        )
        if dupe.first() is not None:
            document.ingest_status = "duplicate"
            await session.flush()
            return

    if not _is_pdf(document):
        document.ingest_status = "unsupported_format"
        await session.flush()
        return

    pdf_bytes = await asyncio.to_thread(downloader, storage_key=document.storage_path)
    pages = extract_pages(pdf_bytes)

    # Decide OCR per page first (so the escape hatch can fire before any writes).
    threshold = settings.tender_ocr_text_density_threshold
    page_payloads: list[tuple[int, str, float | None]] = []  # (page_no, text, ocr_conf)
    ocr_confidences: list[float] = []
    any_ocr = False
    for page in pages:
        if settings.tender_ocr_enabled and is_ocr_candidate(page, threshold=threshold):
            png = render_page_png(pdf_bytes, page_no=page.page_no, dpi=settings.tender_page_render_dpi)
            result = ocr(png)
            any_ocr = True
            ocr_confidences.append(result.mean_confidence)
            page_payloads.append((page.page_no, result.text, result.mean_confidence))
        else:
            page_payloads.append((page.page_no, page.text, None))

    if ocr_confidences:
        mean_conf = sum(ocr_confidences) / len(ocr_confidences)
        if mean_conf < settings.tender_ocr_min_confidence:
            document.ingest_status = "manual_transcription_required"
            document.ocr_applied = True
            await session.flush()
            return

    existing = await session.execute(
        select(TenderPage.page_no).where(TenderPage.document_id == document.id)
    )
    already = {row[0] for row in existing.all()}

    for page_no, text, ocr_conf in page_payloads:
        if page_no in already:
            continue
        png = render_page_png(pdf_bytes, page_no=page_no, dpi=settings.tender_page_render_dpi)
        image_path = (
            f"tender/comparisons/{job.comparison_id}/quotes/{document.quote_id}/"
            f"documents/{document.id}/pages/page-{page_no:04d}.png"
        )
        await asyncio.to_thread(
            uploader, storage_key=image_path, content=png, filename=f"page-{page_no:04d}.png"
        )
        session.add(
            TenderPage(
                document_id=document.id,
                page_no=page_no,
                image_path=image_path,
                text_content=text or "",
                ocr_confidence=ocr_conf,
            )
        )
        await session.flush()  # checkpoint per page

    document.ocr_applied = any_ocr
    document.ingest_status = "ingested"

    quote = await session.get(TenderQuote, document.quote_id)
    if quote is not None:
        quote.stage = "classify_document"
    await jobs.enqueue(
        session,
        kind="classify_document",
        comparison_id=job.comparison_id,
        quote_id=document.quote_id,
        payload={"document_id": str(document.id)},
    )
    await session.flush()

# backend/tender/services/classification.py
"""classify_document stage (PRD §9.1): small-model doc_type adjudication."""

from __future__ import annotations

import uuid

from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from tender.llm.client import TenderLLMClient
from tender.models import DOC_TYPES, TenderDocument, TenderPage, TenderQuote
from tender.schemas import ProjectContext
from tender.services import jobs

PROMPT_VERSION = "0.1.0"


async def classify_document(
    session: AsyncSession,
    job,
    *,
    llm_client: TenderLLMClient | None = None,
) -> None:
    llm_client = llm_client or _default_llm_client()
    document_id = uuid.UUID(job.payload["document_id"])
    document = await session.get(TenderDocument, document_id)
    if document is None:
        raise ValueError(f"classify_document: document {document_id} not found")

    pages = await session.execute(
        select(TenderPage.text_content)
        .where(TenderPage.document_id == document.id)
        .order_by(TenderPage.page_no)
        .limit(2)
    )
    evidence = {
        "filename": document.original_filename,
        "first_pages_text": [row[0] for row in pages.all()],
    }
    context = await _context_for_quote(session, document.quote_id)

    decision = await llm_client.adjudicate(
        "Classify this tender document into one doc_type.",
        list(DOC_TYPES),
        evidence,
        context,
        prompt_version=PROMPT_VERSION,
        model_key="tender_model_adjudicate_small",
    )
    document.doc_type = decision.choice
    document.classification_confidence = decision.confidence

    quote = await session.get(TenderQuote, document.quote_id)
    if quote is not None:
        quote.stage = "extract_line_items"
    await jobs.enqueue(
        session,
        kind="extract_line_items",
        comparison_id=job.comparison_id,
        quote_id=document.quote_id,
        payload={"document_id": str(document.id)},
    )
    await session.flush()


async def _context_for_quote(session: AsyncSession, quote_id: uuid.UUID) -> ProjectContext:
    from tender.services.context import context_for_quote

    return await context_for_quote(session, quote_id)


def _default_llm_client() -> TenderLLMClient:
    from tender.llm.openai_client import AsyncOpenAITenderClient

    return AsyncOpenAITenderClient()

# backend/tender/services/extraction_handler.py
"""extract_line_items worker handler: loads pages, runs the pure extraction
service (reconciliation + confidence gating), persists line items and any
arithmetic-inconsistency flags, then chains to embed_items (PRD §9.2)."""

from __future__ import annotations

import uuid

from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from tender.llm.client import TenderLLMClient
from tender.models import (TenderDocument, TenderFlag, TenderLineItem,
                           TenderPage, TenderQuote)
from tender.schemas import TenderDocumentPage
from tender.services import jobs
from tender.services.context import context_for_quote
from tender.services.extraction import extract_line_items


async def extract_line_items_job(
    session: AsyncSession,
    job,
    *,
    llm_client: TenderLLMClient | None = None,
) -> None:
    llm_client = llm_client or _default_llm_client()
    document_id = uuid.UUID(job.payload["document_id"])
    document = await session.get(TenderDocument, document_id)
    if document is None:
        raise ValueError(f"extract_line_items: document {document_id} not found")
    quote = await session.get(TenderQuote, document.quote_id)
    context = await context_for_quote(session, document.quote_id)

    page_rows = await session.execute(
        select(TenderPage).where(TenderPage.document_id == document.id).order_by(TenderPage.page_no)
    )
    pages = [
        TenderDocumentPage(document_id=str(document.id), page_no=p.page_no,
                           text_content=p.text_content, image_path=p.image_path)
        for p in page_rows.scalars()
    ]

    result = await extract_line_items(
        pages=pages, context=context, llm_client=llm_client,
        stated_total_cents=quote.stated_total_cents if quote else None,
    )

    # Idempotent re-run: clear this document's prior extraction output.
    await session.execute(
        TenderLineItem.__table__.delete().where(TenderLineItem.document_id == document.id)
    )

    for reconciled in result.line_items:
        item = reconciled.item
        session.add(TenderLineItem(
            quote_id=document.quote_id, document_id=document.id, page_no=item.page_no,
            bbox=item.bbox.model_dump() if item.bbox else None,
            description_raw=item.description_raw, section_path=item.section_path or None,
            qty=item.qty, unit=item.unit, rate_cents=item.rate_cents,
            amount_cents=item.amount_cents, item_status=item.item_status,
            allowance_cents=item.allowance_cents,
            extraction_confidence=reconciled.effective_confidence,
        ))

    for flag in result.flags:
        session.add(TenderFlag(
            comparison_id=job.comparison_id, quote_id=document.quote_id,
            flag_type=flag.flag_type, severity=flag.severity,
            headline=flag.headline, detail=flag.detail,
            evidence={"page_no": flag.page_no, "expected_cents": flag.expected_cents,
                      "actual_cents": flag.actual_cents, "delta_ratio": flag.delta_ratio},
        ))

    if quote is not None:
        quote.stage = "embed_items"
    await jobs.enqueue(session, kind="embed_items", comparison_id=job.comparison_id,
                       quote_id=document.quote_id, payload={"document_id": str(document.id)})
    await session.flush()

    return AsyncOpenAITenderClient()
